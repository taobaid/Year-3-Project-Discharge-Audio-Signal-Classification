# 1. Introduction

This repository implements a pipeline for classifying acoustic emissions generated by high-voltage electrical discharge through machine learning models. As high-voltage transmission becomes an increasingly important requirement to the flow of power nationally, it becomes vital to provide a way to analyse transmission line in faults. Analysis of acoustic emissions is a desirable and non-intrusive way to inspect discharges on transmission lines, with particular interest in the application of machine learning models to this analysis.
The system distinguishes between discharge type (conductor, insulator, dry band arcing) and polarity type (AC, +DC, −DC), with 9 machine learning models used overall (ExtraTrees, KNN,  Random Forest, GaussianNB, SVM-RBF, Gradient Boosting, XGBoost, Logistic Regression, MLP), allowing for comparitive assessment for model performance with regards to acoustic analysis. The aim is to assess how each model successfully distinguishes between acoustic signals by discharge type and polarity type.
